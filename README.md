# Portfolio‑Iterations 🎨

A **dynamic, 3‑D portfolio** built with vanilla HTML, CSS and JavaScript.  
The site features:

- **Purple‑glow, glass‑morphism UI** with smooth gradients and custom fonts.  
- **3‑D tilt & cursor‑tracking glow** on each content‑box.  
- **Lazy‑loaded Three.js scenes** in iframes for each iteration (`1.html` – `6.html`).  
- **Ambient synth soundtrack** generated with the Web Audio API.  
- **Nebula‑explosion finale** triggered by the last scroll section (GSAP + Three.js).  

The whole thing is **GitHub‑Pages ready** – just push the repo and enable Pages.
